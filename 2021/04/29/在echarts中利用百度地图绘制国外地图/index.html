如何在echarts中利用百度地图api绘制国外地图

申请百度地图的JavaScript API

百度地图JavaScript API是一套由JavaScript语言编写的应用程序接口，可帮助您在网站中构建功能丰富、交互性强的地图应用，支持PC端和移动端基于浏览器的地图应用开发，且支持HTML5特性的地图开发。本项目使用的是v3.0版本。
